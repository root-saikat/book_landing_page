Foreword Of The Book Author

Saikat Cullen @SaikatCullen

"Tailwind really allows you to build a unique website design and without you needing to write any CSS code. Maybe you'll think that it will take you longer when building a website with a utility-first framework than with the UIkit framework. Yes, that's right. Because you have to think about the design and implement the design from scratch yourself. However, you need to keep in mind the benefits provided by the utility-first framework, so you can consider them. In this book, we'll study the Tailwind CSS framework. Starting from the concept, workflow, to creating a responsive website with TailwindCSS. This book is not for everyone, at least you understand how to operate a computer, understand HTML, understand CSS, or you can say you have built a website before and want to learn new things to improve your skills."
